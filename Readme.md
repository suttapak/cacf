### Install & Usage.

##### **Pre Requirements**

- [GO](https://go.dev/)
- [Docker](https://www.docker.com/)

```bash

git clone https://github.com/suttapak/cacf.git
cd cacf

sudo docker compose up -d

go mod tidy
go run main.go

```
